# Пиар-помощник (VK PR Helper)

Веб-сервис для ролевиков ВКонтакте, позволяющий создавать текстовые шаблоны объявлений и отправлять предложенные посты в ролевые сообщества.

## Стек технологий
- **Frontend**: React, Vite, TypeScript (папка `packages/web`)
- **Backend**: Node.js, Express, Yandex Cloud Functions (папка `yc`)
- **База данных**: Yandex Managed Service for YDB (Serverless)
- **Хранилище**: Yandex Object Storage (S3)

## Как устроен проект

- Проект спроектирован для развертывания в **Yandex Cloud**. 
- Бэкенд представляет собой Serverless-функцию, взаимодействующую с YDB.
- Фронтенд - это SPA приложение, которое хостится в Object Storage (S3).
- В проекте строго соблюдается ФЗ-152: токены доступа не логируются, все ID пользователей ВКонтакте хешируются перед записью в базу данных.

## Запуск (для ознакомления с кодом)

Для полноценного локального запуска потребуется развернутая инфраструктура в Yandex Cloud (база данных YDB, Lockbox для секретов, S3-бакеты). 

Для установки зависимостей (требуется `pnpm`):
```bash
pnpm install
```

Сборка фронтенда:
```bash
pnpm --filter web build
```

Сборка бэкенда:
```bash
npx esbuild yc/index.ts --bundle --platform=node --target=node22 --outfile=dist/index.js
```

Переменные окружения, необходимые для работы (см. `.env.example`), должны быть предоставлены вашим Yandex Cloud профилем. Секретный сервисный ключ ВК загружается из Yandex Lockbox.
